# Christopher Wampler

I build systems that solve real problems for real people.

Currently focused on **AI-integrated applications**, **automation**, and **full-stack product development** — with a bias toward things that actually ship and run in production.

## What I'm Building

**[Slarti](https://github.com/ChristopherWampler/Slarti)** — A compound AI system built as a family garden companion. Multi-provider AI routing (Claude, Gemini, OpenAI), pgvector semantic memory, Discord + voice integration, automated weather monitoring, and a regional knowledge engine with 390+ indexed sources from MU Extension and the Farmer's Almanac. Runs in production on a home server with 6 automated agents on cron.

**[Homeschool App](https://github.com/ChristopherWampler/Homeschool_App)** — Full-stack PWA for homeschool administration. React 18 / TypeScript frontend, PostgreSQL backend, n8n automation, offline-first architecture, AI-powered weekly insights. Built for Missouri state compliance. Design principle: *if it's not easier than Excel, it failed.*

## How I Think

I care about **systems design** — how components fit together, how data flows, how things fail gracefully. I document decisions, not just code. I build for maintainability because I'm usually the one maintaining it at 3 AM.

I'm drawn to problems where the solution isn't another SaaS dashboard but a thoughtful integration of real tools into real workflows.
